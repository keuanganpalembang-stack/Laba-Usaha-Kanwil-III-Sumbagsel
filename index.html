📖 Cara Upload / Update Data Excel
1
Siapkan file Excel bulan baru

Pastikan format sheet LABA BOPO PER OUTLET sama dengan bulan sebelumnya (kolom tidak berubah posisi).

2
Jalankan script konversi Python

Buka terminal, pastikan Python 3 + pandas sudah terinstall, lalu jalankan:

python convert.py --bulan juni --file "LABA_Juni_2026.xlsx"

Script akan menghasilkan file data/juni.json secara otomatis.

3
Tambahkan opsi bulan baru di index.html

Buka index.html, cari tag <select id="monthSelect">, tambahkan baris:

<option value="juni">Juni 2026</option>
4
Push ke GitHub 
git add data/juni.json index.html
git commit -m "Tambah data Juni 2026"
git push

Dashboard otomatis update — tidak perlu rebuild apapun.

💡 Tips: File JSON di folder data/ adalah satu-satunya yang perlu diganti setiap bulan. Kode app.js dan style.css tidak perlu disentuh.
